# K-Means Clustering Tutorial (Mall Customer Dataset)

## Overview
This project is a tutorial on K-Means clustering, an unsupervised machine learning technique used to group similar data points. The project demonstrates how clustering works using the Mall Customer dataset.

## Objective
- Understand how K-Means clustering works
- Visualize cluster formation
- Analyze the impact of choosing the number of clusters (K)
- Use the Elbow Method to determine optimal K

## Dataset
Mall Customer Dataset containing:
- Age
- Gender
- Annual Income (k$)
- Spending Score (1-100)

For clustering, the following features are used:
- Annual Income (k$)
- Spending Score (1-100)

## Methods Used
- Data preprocessing
- Feature scaling (StandardScaler)
- K-Means clustering
- Elbow method
- Cluster visualization
